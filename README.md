# 💊 Supplement Sales Dashboard | Power BI

## 📌 Project Overview

The **Supplement Sales Dashboard** is an interactive Power BI project designed to analyze sales performance, customer purchasing trends, product categories, and revenue generation in the dietary supplement industry.

The dashboard transforms raw sales data into actionable business insights, enabling stakeholders to make informed decisions regarding product performance, inventory planning, and sales strategies.

---

## 🎯 Objectives

* Analyze overall sales performance.
* Identify top-performing products and categories.
* Track revenue trends over time.
* Monitor regional sales distribution.
* Evaluate discount effectiveness.
* Analyze product return patterns.
* Compare sales across different platforms.

## 🗂 Dataset Information

The dataset contains weekly supplement sales records with the following attributes:

| Column Name    | Description          |
| -------------- | -------------------- |
| Date           | Sales Date           |
| Product Name   | Supplement Product   |
| Category       | Product Category     |
| Units Sold     | Number of Units Sold |
| Price          | Product Price        |
| Revenue        | Generated Revenue    |
| Discount       | Applied Discount     |
| Units Returned | Returned Units       |
| Location       | Sales Region         |
| Platform       | Sales Platform       |
